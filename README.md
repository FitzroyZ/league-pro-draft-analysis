# Project Overview: Tactical Insights & Analytics Platform

As a data professional with extensive experience in managing BI platforms, I recognized that standard analytics tools often fall short when addressing the granular needs of match analysis and team-specific statistics. I built this project to bridge that gap, providing coaching staffs with actionable insights—especially during high-stakes events like the World Championship—to ensure data-driven confidence when it matters most.

# Core Project Features

*   **Precision-Focused Data**: Curated data covering 2025 and 2026, specifically isolated to account for the unique draft rules of the current era.
*   **Multi-Dimensional Filtering**: Advanced query capabilities across Year, Patch, Side, FirstPick, Team, and Player metrics.
*   **1st Round B/P Deep-Dive**: Dedicated analysis on the initial draft phase, identifying the specific correlations between early picks/bans and final match outcomes.
*   **Strategic Decision Support**: A high-utility interface designed to transform complex raw data into clear, decision-ready visual references for coaching teams.
